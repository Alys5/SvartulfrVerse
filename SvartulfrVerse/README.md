# SvartúlfrVerse

## Repository Purpose

SvartúlfrVerse is the active runtime canon repository for the character simulation system. This repository serves as the single source of truth for validated canonical content and runtime behavior.

## Current Scope

This repository is intentionally scoped to:

- **Only Human** - No supernatural elements, no fantasy races, no cyber-enhancements
- **Contemporary** - Modern real-world setting only
- **Los Angeles Dynasty** - Bloodmoon Dynasty, Douglas Dynasty, and Douglas-Bloodmoon line

All other content types (Urban Fantasy, Cyber, Wasteland, Fantasy, Norse Mythic, Regency, Overlay Systems, Pack Systems, Supernatural Systems) are explicitly out of scope and remain in the historical archive until validated for reintroduction.

## Rebuild Status

**Phase: Bootstrap**

This repository is currently in bootstrap phase. No runtime logic, characters, worlds, or lorebooks are implemented yet. The repository contains only architectural documentation and engine layer placeholders.

## Canon Recovery Workflow

This repository follows a strict canon recovery workflow:

1. NotebookLM Discovery (Secondary Evidence)
2. Historical Archive Verification
3. Character Audit
4. Architecture Review
5. Canon Decision
6. Repository Import

No content is imported from the historical archive without audit validation.
