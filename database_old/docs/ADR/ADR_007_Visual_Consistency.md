# ADR 007: Visual Consistency

## Context

A coherent visual identity is required to maintain immersion and brand recognition across the SvartúlfrVerse.

Generic AI aesthetics produce visual drift and weaken world identity.

## Decision

All visual content follows a hierarchical inheritance system.

Global DNA
↓
World DNA
↓
Overlay DNA
↓
Character Identity Anchors

## Global Visual DNA

The baseline aesthetic.

### Lighting

- Warm cinematic lighting
- Rembrandt lighting
- Deep shadows
- Golden hour tones

### Palette

- Amber
- Gold
- Obsidian
- Mahogany
- Slate Grey

### Style

- Semi-realistic
- Painterly realism
- Luxury visual storytelling

## World Visual DNA

Defines:

- architecture
- environments
- atmosphere
- cultural visual language

## Overlay Visual DNA

Adds additional elements.

Never replaces parent World DNA.

### Example

Urban Fantasy inherits Contemporary.

Adds:

- Kemonomimi
- Fangs
- Hidden supernatural society
- Moonlit symbolism

## Character Identity Anchors

Characters always retain their visual identity.

Character Anchors have higher priority than World DNA.

### Example

Erik

Always:

- black hair with grey streaks
- amber eyes
- intimidating corporate monarch

Never:

- silver-haired elder king

### Example

Wulfnic

Always:

- ancient patriarch
- pale blond to white hair
- old-world aristocracy

Never:

- modern CEO aesthetic

## Visual Pipeline

Reference Library
↓
Moodboards
↓
Visual DNA
↓
ComfyUI Workflow
↓
Generated Assets

## Banned Elements

- Generic anime
- Flat photorealism
- Pastel palettes
- Generic cyberpunk neon
- Inconsistent character appearance

## Consequences

Every World requires:

- Visual_DNA.md
- Reference assets
- Moodboards

Every Character requires Identity Anchors.

Visual consistency becomes enforceable.
