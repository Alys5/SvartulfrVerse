## Purpose

Adapt the SvartúlfrVerse setting into a large-scale interstellar civilization while preserving core character and dynastic structures.

This concept transposes the Douglas-Bloodmoon dynastic narrative into a futuristic space opera setting, maintaining the family dynamics, power structures, and character relationships while reimagining the worldbuilding for a galactic scale.

## Design Principles

1. **Preserve character identities** — Core personalities, relationships, and family dynamics remain intact
2. **Scale up the world** — Replace terrestrial settings with interstellar equivalents
3. **Maintain dynastic tension** — The Douglas-Bloodmoon power dynamic translates to galactic corporate/military rivalry
4. **Keep the tone** — Corporate dynasty, high society, urban luxury → Galactic aristocracy, megacorporate power, station luxury
