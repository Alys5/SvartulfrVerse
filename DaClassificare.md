- `Info Board [Large].txt`: template UI per `<info_board>` con campi data, ora, meteo, location, mood, thought, aff/arousal, outfit, goal.
- `Info Board [Small].txt`: template UI per `<Info_Board>` con label tempo corrotta nel file sorgente.

- `[ib]_succ_theme_[small].json`: script tema CSS/regex per Info Board Small; risulta disabilitato nel JSON e presenta encoding corrotto su una label.
- `SUCC University.json`: configurazione tema UI per chat/ambiente SUCC.
