## Purpose

This document defines the canonical visual baselines for the Los Angeles Dynasty. All visual characteristics flow from these baselines.

---

## Visual Baseline Definitions

### Douglas Visual Baseline

**Reference Character:** Erik Douglas

| Attribute | Value | Status |
|-----------|-------|--------|
| Hair Color | Black (streaked with silver/grey) | Canonical |
| Eye Color | Amber | Canonical |
| Build | Massive, muscular, imposing | Canonical |
| Height | 205cm | Canonical |
| Skin | Weathered, mature | Canonical |
| Aesthetic | Corporate monarch, power suits | Canonical |

**Visual DNA Anchor:**
```
Erik Douglas:
  Always:
    - Black hair streaked with silver/grey, slicked back
    - Amber eyes, cold and unyielding
    - 205cm, massive muscular build, weathered skin
    - Corporate monarch aesthetic, bespoke power suits
  Never:
    - Long white hair
    - Ancient king aesthetic
    - Nordic patriarch styling or fur robes
    - T-shirts or casual wear
```

### Bloodmoon Visual Baseline

**Reference Character:** Wulfnic Bloodmoon

| Attribute | Value | Status |
|-----------|-------|--------|
| Hair Color | Silver-white | Canonical |
| Eye Color | Silver-white (glowing in low light) | Canonical |
| Build | Lean, strong, refined | Canonical |
| Height | 195cm | Canonical |
| Skin | Pale, ancient | Canonical |
| Aesthetic | Ancestral nobility, old-world aristocratic | Canonical |

**Visual DNA Anchor:**
```
Wulfnic Bloodmoon:
  Always:
    - Silver-white hair, long and braided with silver
    - Silver-white eyes, glowing faintly in low light
    - 195cm, lean but impossibly strong, ancient skin
    - Ancestral nobility aesthetic, heavy furs
  Never:
    - CEO aesthetic
    - Executive skyscrapers or corporate boardrooms
    - Modern streetwear or tactical gear
```

## Inheritance Context

These baselines serve as the reference points for the Fusion Inheritance Model (see [Inheritance_Rules.md](./Inheritance_Rules.md)).

Children of Erik Douglas + Nixara Bloodmoon inherit visual characteristics according to fusion rules:
- **Dominant:** Douglas (black hair, amber eyes, massive build)
- **Recessive:** Bloodmoon (silver-blonde hair, pale/blue eyes, lean build)
