# Bot Visual Package Standard v1

---

## Purpose

Every bot must include a standardized visual package. The goal is to:
- increase click-through rate
- improve immersion
- establish immediate character identity
- visually communicate personality
- maintain brand consistency across the entire SvartúlfrVerse ecosystem

---

# REQUIRED IMAGE PACKAGE

Each bot must include:

| # | Image Type | Format | Purpose |
|---|------------|--------|---------|
| 01 | Primary Portrait | 1:1 | Identity anchor — first impression |
| 02 | Lifestyle Portrait | 4:3 | Everyday life atmosphere |
| 03 | Signature Activity Scene | 5:3 | What makes this character special |
| 04 | Emotional Scene | 4:3 | Emotional depth — the mask comes off |
| 05 | Environment Banner | 3:1 | World establishment |

**Total: 5 images minimum**

---

# IMAGE 01 — PRIMARY PORTRAIT

**Purpose:** Identity anchor. This is the image users see first.

**Requirements:**
- direct eye contact
- waist-up portrait
- clean composition
- character centered
- signature expression
- recognizable accessories

**Format:** 1:1

**Usage:** Bot avatar, Card thumbnail, Search results

---

# IMAGE 02 — LIFESTYLE PORTRAIT

**Purpose:** Show the character's everyday life.

**Question answered:** "What does spending time with this person feel like?"

**Requirements:**
- relaxed pose
- natural environment
- casual clothing
- authentic smile
- slice-of-life atmosphere

**Format:** 4:3

**Examples by archetype:**
- Student: studying in a café
- Artist: painting outdoors
- Programmer: working on a laptop
- Athlete: training in a gym

---

# IMAGE 03 — SIGNATURE ACTIVITY SCENE

**Purpose:** Show the thing the character is known for.

**Question answered:** "What makes this character special?"

**Requirements:** Character actively performing their defining activity.

**Format:** 5:3

**Examples by character:**
- Alyssa: community event / art studio / hospital volunteering
- Jasper: parkour rooftop run / DJ set / coding in the Blackroom
- Malachia: boxing training / MMA ring / gym
- Noah: charity gala / law library / baking in kitchen
- Erik: UCLA football game / DCC boardroom / estate garden
- Logan: working on a car engine / behind The Verve bar
- Wulfnic: reading in library / passing down a pendant

---

# IMAGE 04 — EMOTIONAL SCENE

**Purpose:** Show emotional depth.

**Question answered:** "What happens when the mask comes off?"

**Requirements:** Strong emotional storytelling.

**Possible emotions:** vulnerable, reflective, lonely, determined, hopeful, grieving, protective

**Format:** 4:3

**This image should feel like a movie still.**

---

# IMAGE 05 — ENVIRONMENT BANNER

**Purpose:** Establish setting.

**Question answered:** "What world does this character belong to?"

**Requirements:**
- Environment-focused image
- Character may be absent or very small

**Format:** 3:1

**Examples:**
- Douglas Estate (Beverly Hills compound)
- UCLA Campus (Westwood)
- The Verve (Arts District DTLA)
- Douglas Customs (Logan's workshop)
- Wulfnic's Library
- Santa Monica Waterfront
- Los Angeles Skyline

---

# STYLE CONSISTENCY RULES

All images must use:
```
MASTERPIECE, BEST QUALITY, SEMI-REALISTIC NIJI STYLE,
PREMIUM VISUAL NOVEL ARTWORK, PAINTERLY REALISM,
ANIME-INSPIRED REALISM, CALIFORNIA SLICE-OF-LIFE DYNASTY,
GOLDEN HOUR SUNLIGHT, WARM NATURAL LIGHTING,
EMOTIONAL REALISM, HIGHLY DETAILED FACES
```

**No image may deviate from the approved visual authority.**

---

# CHARACTER RECOGNITION RULE

A user must be able to identify the character immediately even if:
- clothing changes
- location changes
- camera angle changes

**Recognition must come from:**
- face
- hairstyle
- eyes
- body language
- iconic accessories
- color palette

**Never rely exclusively on clothing.**

---

# VISUAL STORYTELLING RULE

The 5 images together must answer:

1. **Who is this?** (Primary Portrait)
2. **What are they like?** (Lifestyle Portrait)
3. **What do they do?** (Signature Activity)
4. **What do they feel?** (Emotional Scene)
5. **What world do they live in?** (Environment Banner)

If one of these questions is not visually answered, the package is incomplete.

---

# EXTENDED PACKAGE (7 IMAGES) — TIER-1 CHARACTERS

For Tier-1 characters (Alyssa, Jasper, Noah, Malachia, Erik, Logan, Wulfnic), use the extended 7-image standard:

| # | Image Type | Format | Purpose |
|---|------------|--------|---------|
| 01 | Primary Portrait | 1:1 | Identity anchor |
| 02 | Casual Lifestyle | 4:3 | Everyday atmosphere |
| 03 | Signature Activity | 5:3 | Defining trait |
| 04 | Emotional Scene | 4:3 | Emotional depth |
| 05 | Relationship Scene | 4:3 | Connection with another character |
| 06 | Full Body Character Sheet | 4:3 | Complete visual reference |
| 07 | Environment Banner | 3:1 | World establishment |

**Image 05 — Relationship Scene:** Character interacting with a key relationship (family member, partner, friend). Shows dynamic, not just individual.

**Image 06 — Full Body Character Sheet:** Standing full-body pose, neutral background, canonical outfit. Used for reference and consistency.
